Short Description An adversary runs attacker-controlled code on the target system by exploiting Remote Services: Windows Remote Management. Windows Security Event 4624 (An account was successfully logged on) provides the primary detection signal when properly audited and monitored in a SIEM.

2. MITRE ATT&CK Mapping

  • Tactic : Execution

Technique : Remote Services: Windows Remote Management (T1021.006)

3. How the Attack Typically Works (Procedure)

  1. Attacker authenticates to the target host using stolen credentials (pass-the-hash or plaintext).
  2. WMI queries or Win32_Process create calls are issued remotely via the DCOM/RPC transport on port 135 and dynamic ports.
  3. The remote WMI call spawns a new process on the target — visible as Event 4688 with parent WmiPrvSE.exe or svchost.exe (WMI service).
  4. The spawned process executes the attacker's command (PowerShell download cradle, reverse shell, or reconnaissance command).
  5. Event 4624 LogonType 3 with AuthenticationPackageName indicating NTLM or Kerberos precedes the WMI process creation.
Most common tools observed in 2024–2026:
  • Impacket wmiexec.py
  • CobaltStrike WMI lateral movement
  • PowerShell Invoke-WmiMethod
  • WMIS / wmic.exe (native)
  • Metasploit windows/wmi/wmi_exec

5. Detection / Hunting Recommendations

Strong / Classic Signature Rule (High Confidence)
  • Alert on Event ID 4624 matching the highest-fidelity field combination shown above
  • Apply thresholds appropriate to the environment: a single event may be sufficient for critical-level detections; volume thresholds apply for medium/low
  • Correlate with Event 4624 (successful logon) using SubjectLogonId to establish the session context and identify the source IP and workstation
Highest-Fidelity Correlation (SIEM Query Logic): 
EventID = 4624
AND SubjectUserName NOT IN <approved_admin_accounts>
AND IpAddress NOT IN <trusted_ip_ranges>
AND LogonType IN (3, 10)
Contextual / Behavioral Hunting Queries (SIEM / EDR)
  • Correlate 4624 events where SubjectLogonId links to a Type 3 (network) or Type 10 (remote interactive) logon from an external IP address
  • Alert on 4624 events from accounts in privileged groups (Domain Admins, Enterprise Admins) outside approved maintenance windows
  • Hunt for 4624 events followed within 60 seconds by lateral movement indicators (5140, 4624 to other hosts) from the same SubjectLogonId
False Positive Considerations
  • Legitimate after-hours work by on-call staff, developers, or global teams in different time zones
  • Service accounts performing scheduled maintenance tasks outside business hours
  • Automated monitoring and backup systems generating logon events at any hour
Recommended Hardening (Prevention & Impact Reduction)
  • Implement privileged access workstations (PAW) to restrict administrative logons to dedicated secure hosts
  • Enforce MFA for all remote and VPN logons to prevent credential-only attacks
  • Configure Conditional Access policies to block high-risk logons based on impossible travel or unfamiliar devices
  • Deploy LAPS to prevent lateral movement using shared local administrator credentials

6. Recommended Response Actions

  • Isolate the affected host from the network immediately if active exploitation or lateral movement is confirmed.
  • Identify the initiating account (SubjectUserName/SubjectLogonId) and disable it pending investigation.
  • Preserve all relevant Windows Security Event logs from the affected host(s) before any remediation.
  • Correlate the triggering event with Event 4624 logon records to identify the source IP, workstation, and session that performed the action.
  • Identify all systems the attacker accessed from the compromised host by reviewing outbound connection logs and Event 5140/4624 entries.
  • Reset credentials for all accounts used for lateral movement; review and revoke any new accounts or group memberships created.
  • Scan all reached hosts for persistence mechanisms (registry, scheduled tasks, services, WMI subscriptions).
  • Implement network segmentation controls to prevent unrestricted lateral movement on the same detection path.
